Наборы данных car_data.csv car_data_addit.json предназначены для прогнозирование продажной цены подержанных автомобилей.

car_data.csv содержит основные данные об автомобилях:

1) car_id - id номер автомобиля.
2) Car_Name - указывает модель или название автомобиля
3) Year - обозначает год выпуска автомобиля.
4) Present_Price - цена, по которой на данный момент продается автомобиль.
5) Selling_Price - фактическая цена, по которой был продан автомобиль.
6) Kms_Driven - обозначает общее пройденное расстояние в километрах.

car_data_addit.json содержит дополнительные данные об автомобилях:

1) car_id - id номер автомобиля.
2) Car_Name - указывает модель или название автомобиля.
3) Fuel_Type - указывает тип топлива автомобиля (например, бензин, дизель, гибрид, электричество).
4) Seller_Type - указание того, является ли продавец физическим лицом или дилером.
5) Transmission - указывает тип трансмиссии автомобиля (например, механическая, автоматическая).
6) Owner - указание того, были ли у автомобиля предыдущие владельцы.

Тип SQL - SQLite
